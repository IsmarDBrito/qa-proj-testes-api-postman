## Objetivo
Avaliar o estado geral de um aplicativo (app) e utilizar dados brutos (formato XLSX) para suportar a conclusão da avaliação, demonstrando capacidade analítica e de síntese de resultados.

## Habilidades de QA Demonstradas
* **Análise de Dados Brutos:** Manipulação de dados de teste para extração de métricas de qualidade.
* **Relatório de Avaliação:** Elaboração de um documento sumário que consolida achados e avalia a funcionalidade do app.
* **Comunicação Técnica:** Habilidade em traduzir dados brutos em conclusões de alto nível para stakeholders.

## Conteúdo do Repositório
* `dados_brutos_para_analise.xlsx`: Conjunto de dados brutos utilizados como base para a avaliação.
* `relatorio_avaliacao_funcional.txt`: Sumário da avaliação e conclusões.
